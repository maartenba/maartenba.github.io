[![](/images/WindowsLiveWriter/Articlein.NETmagazine_9C2A/20061226-netmag_thumb.jpg)](/images/WindowsLiveWriter/Articlein.NETmagazine_9C2A/20061226-netmag.jpg) The new [.NET magazine Belgium/Netherlands](http://www.microsoft.com/netherlands/msdn/netmagazine/default.aspx) has been distributed to all subscribers. For the latest edition, [#15](http://www.microsoft.com/netherlands/msdn/netmagazine/code/magazine15.aspx), I've written an article on the basic principles and usage of [NHibernate](http://www.nhibernate.org), an ORM-mapper for the .NET platform. It's written in Dutch, so I must disappoint my English blog readers...

A PDF version and example code can be downloaded on the [.NET magazine #15 homepage](http://www.microsoft.com/netherlands/msdn/netmagazine/code/magazine15.aspx). Comments can be posted on this blog post.
