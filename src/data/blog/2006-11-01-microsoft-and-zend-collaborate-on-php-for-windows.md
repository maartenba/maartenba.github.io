A nice press release today at Microsoft: [Microsoft and Zend Technologies Announce Technical Collaboration to Improve Interoperability of PHP on the Windows Server Platform](http://www.microsoft.com/presspass/press/2006/oct06/10-31MSZendPR.mspx). Both companies admit that PHP is often developed on Windows systems, but the final server environment is mostly *nix. With this collaboration, [Microsoft](http://www.microsoft.com) and [Zend](http://www.zend.com) will try to change that.

The specific points of action in this collaboration:

- Microsoft plans to releas an IIS add-on, FastCGI, that will serve as the interface between PHP and IIS. I talked about [FastCGI earlier](http://www.balliauw.be/maarten/blog/16,php-on-iis7-rc-1-but-unsafe.htm).
- Zend will establish a Windows testing lab, to maintain high performance of PHP on Windows servers as PHP evolves.
- Both parties will help to ensure a production-quality PHP environment for IIS 5.1 and newer versions
- Both parties will provide support to the PHP community, helping developers planning to deploy PHP applications on Windows servers.


I think this is nice news! Looking forward to results of this collaboration...
