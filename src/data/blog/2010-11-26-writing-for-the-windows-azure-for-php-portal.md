I actually just noticed it has been a while since I did a blog post. I also know that writing about this is [not really a good idea in the blogosphere](http://www.hanselman.com/blog/BlogInteresting32WaysToKeepYourBlogFromSucking.aspx). Unless… it’s for a good reason!


The good reason for not being that active on my blog lately is the fact that I’m producing content for [Microsoft’s Interoperability team](http://www.interoperabilitybridges.com). Have you ever wanted to start working with [Windows Azure](http://www.azure.com) and [PHP](http://www.php.net)? No idea where to start? Meet the official portal: [Developing Applications for Azure with PHP](http://azurephp.interoperabilitybridges.com/).


I’ve currently posted some tutorials and scenarios out there, but there’s more to come. Here’s a list of what’s currently available:


- [Tutorial - Using the Windows Azure Tools for Eclipse with PHP](http://azurephp.interoperabilitybridges.com/articles/tutorial-using-the-windows-azure-tools-for-eclipse-with-php)
- [Tutorial - Using the Windows Azure SDK for PHP](http://azurephp.interoperabilitybridges.com/articles/tutorial-using-the-windows-azure-sdk-for-php)
- [Tutorial - Using Queue Service](http://azurephp.interoperabilitybridges.com/articles/tutorial-using-queue-service)
- [Scenario - Offloading static content to blob storage](http://azurephp.interoperabilitybridges.com/articles/scenario-offloading-static-content-to-blob-storage)


So whenever you think I’m relaxing and doing nothing, check [http://azurephp.interoperabilitybridges.com](http://azurephp.interoperabilitybridges.com) for new content. By the way, if you are doing PHP and Azure, drop me a line. It’s always good to know and maybe I can be of help.


Stay tuned for more on this!
