![image](/images/WindowsLiveWriter/OpenXMLsupportinZendFrame.7Luceneindexer_1377E/image_d7497c6a-0c23-477d-80d0-cea7d4f68856.png) It has indeed been a long time since I blogged about OpenXML. However, this does not mean I'm completely doing nothing with it! [PHPExcel](http://www.phpexcel.net) still keeps eating a lot of evening hours as well as some other OpenXML-related projects.

The folks at [Zend](http://www.zend.com) contacted me if I would be interested in implementing OpenXML indexers for the Zend Framework Lucene Indexer. I've blogged about [indexing OpenXML files](/post/2008/02/01/Indexing-Word-2007-(docx)-files-with-Zend_Search_Lucene.aspx) once before, but since today's [Zend Framework 1.7 release](http://devzone.zend.com/article/4045-Zend-Framework-1.7.0-is-now-available), indexing OpenXML documents is as easy as indexing any other natively supported file format. Want to know how? Check the documentation in the manual at [http://framework.zend.com/manual/en/zend.search.lucene.html](http://framework.zend.com/manual/en/zend.search.lucene.html).
