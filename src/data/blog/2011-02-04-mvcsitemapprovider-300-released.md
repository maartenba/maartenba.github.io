[![](http://download.codeplex.com/Project/Download/FileDownload.aspx?ProjectName=mvcsitemap&DownloadId=137766&Build=17501)](http://mvcsitemap.codeplex.com/)Just a quick note to tell that MvcSiteMapProvider 3.0.0 has been released and is available on [CodePlex](http://mvcsitemap.codeplex.com/releases) and [NuGet](http://www.nuget.org/Packages/Packages/Details/MvcSiteMapProvider-MVC3-3-0-0-0). This version is targeted at ASP.NET MVC 3. If you are still using ASP.NET MVC 2, no worries: version 2.3.1 contains the exact same functionality but is targeted at ASP.NET MVC 2.


The changelog:


- [Additional UpdatePriority options for generating XML sitemaps](http://mvcsitemap.codeplex.com/workitem/5701)
- [Allow to specify target on SiteMapTitleAttribute](http://mvcsitemap.codeplex.com/workitem/5649)
- [One action with multiple routes and breadcrumbs](http://mvcsitemap.codeplex.com/workitem/5640)
- [Medium Trust optimizations](http://mvcsitemap.codeplex.com/workitem/5628)
- [Create SiteMapTitleAttribute for setting parent title](http://mvcsitemap.codeplex.com/workitem/5574)
- [IntelliSense for your sitemap with MvcSiteMapSchema.xsd](http://mvcsitemap.codeplex.com/workitem/5571)
- [Change skipAssemblyScanOn to excludeAssembliesForScan and includeAssembliesForScan](http://mvcsitemap.codeplex.com/workitem/5494)
- Razor templates in the NuGet package


Enjoy!
