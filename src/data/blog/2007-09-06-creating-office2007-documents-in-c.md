Here are some additional links to get you started:

- [OpenXML ECMA specifications](http://www.ecma-international.org/news/TC45_current_work/TC45_available_docs.htm) (the complete OpenXML specs, a HUGE set of documents explaining almost everything)
- [OpenXML explained](http://openxmldeveloper.org/articles/1970.aspx) ([Wouter's](http://blogs.infosupport.com/wouterv) book on OpenXML, actually the basics of the ECMA specs you will be confronted with while developing)
- [OpenXMLdeveloper.org](http://openxmldeveloper.org/) (forums, how-to's, ...)
- [PackageExplorer](http://www.codeplex.com/PackageExplorer/) ([Wouter's](http://blogs.infosupport.com/wouterv) way to validate your OpenXML document, and browse trough all parts inside an OpenXML package - it also includes a cool start page created by a [Belgian developer](/) [<:o)])
- [Microsoft Office2007 compatibility pack](http://www.microsoft.com/downloads/details.aspx?familyid=941b3470-3ae9-4aee-8f43-c6bb74cd1466&displaylang=en) for OfficeXP and Office 2003
- Videos of the worldwide [OpenXML workshops](http://openxmldeveloper.org/archive/2007/08/29/2066.aspx) by [Doug](http://blogs.msdn.com/dmahugh) and several others
- [Brian Jones' blog](http://blogs.msdn.com/brian_jones), Office rogram manager
- A small [Wordpad-alike C# project](http://www.openxml.biz/OpenXMLWriter.html) spitting out WordML, source code included!
