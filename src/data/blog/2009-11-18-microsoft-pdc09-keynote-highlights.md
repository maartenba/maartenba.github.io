Finally found some time to write a short blog post on the announcements this morning at [PDC 2009](http://www.microsoftpdc.com).[![](http://blogs.msdn.com/blogfiles/msftisvs/WindowsLiveWriter/DontMissPDC09fortheFutureofDeveloperTech_1386F/PDC09Bling_BeforeAfter_240_2.jpg)](http://www.microsoftpdc.com) Ray Ozzie started the keynote this morning, focusing on Microsoft’s “three-screen” vision for the future. There will be three screens connected to the cloud: TV, (handheld) devices and of course good old PC. This vision is driven by some key players: Windows 7, Internet Explorer, Silverlight and Windows Azure. Make sure to have a look at these four if you want to play in this future.


Some announcements were made as well:


- [Microsoft Codename “Dallas”](http://www.pinpoint.com/en-US/Dallas), a new approach to consuming data that is publicly available
- [Microsoft Pinpoint](http://pinpoint.microsoft.com/en-US/default.aspx), a marketplace for applications on Windows Azure
- [Windows Server AppFabric](http://msdn.microsoft.com/lt-lt/windowsserver/ee695849(en-us).aspx), the on-premise extension to the Azure platform
- [Windows Identity Foundation](http://msdn.microsoft.com/en-us/security/aa570351.aspx), formerly known as Geneva, has gone RTW today
- [Download ASP.NET MVC 2 Beta](http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=4817cdb2-88ea-4af4-a455-f06b4c90fd2c&utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+MicrosoftDownloadCenter+%28Microsoft+Download+Center%29&utm_content=Google+Reader#tm), released today
- [Download SQL Server Modeling CTP](http://feeds.feedburner.com/SQL%20Server%20Modeling%20CTP%20-%20Nov%202009), the repository part from Oslo
- [Download Microsoft Sync Framework Power Pack for SQL Azure November CTP](http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=bce4ad61-5b76-4101-8311-e928e7250b9a&utm_source=feedburner&utm_medium=feed&utm_campaign=Feed%3A+MicrosoftDownloadCenter+%28Microsoft+Download+Center%29#tm), a cool synchronization means between local, on-premise data and data in SQL Azure. Yummy!
