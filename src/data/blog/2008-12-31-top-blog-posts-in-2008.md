![Happy new year 2009!](/images/WindowsLiveWriter/Topblogpostsin2008_D71D/image_5.png) Since everyone (that is [Simone Chiaretta](http://codeclimber.net.nz/archive/2008/12/31/best-of-2008-5-most-popular-posts.aspx) and [Phil Haack](http://www.haacked.com/archive/2008/12/30/not-your-typical-top-ten-of-2008-post.aspx)) seems to be blogging about 2008 and their top posts, here’s my list.

## Top 5 blog posts in 2008

**1.** [Reuse Excel business logic with PHPExcel](/post/2008/03/27/reuse-excel-business-logic-with-phpexcel.aspx) – This post explains how to use PHPExcel in a business scenario where a spreadsheet contains calculation logic which is used in a PHP application.

**2.** [Code performance analysis in Visual Studio 2008](/post/2008/02/07/code-performance-analysis-in-visual-studio-2008.aspx) - Visual Studio developer, did you know you have a great performance analysis (profiling) tool at your fingertips?

**3.** [ASP.NET load balancing and ASP.NET state server (aspnet_state)](/post/2007/11/22/ASPNET-load-balancing-and-ASPNET-state-server-(aspnet_state).aspx) – A how-to on load balancing with ASP.NET. Also related to [ASP.NET Session State Partitioning](/post/2008/01/23/ASPNET-Session-State-Partitioning.aspx) and [ASP.NET Session State Partitioning using State Server Load Balancing](/post/2008/01/24/ASPNET-Session-State-Partitioning-using-State-Server-Load-Balancing.aspx).

**4.** [Code based ASP.NET MVC GridView](/post/2008/06/04/code-based-aspnet-mvc-gridview.aspx) – This post gives a shot at a gridview for the ASP.NET MVC framework, built using expressions.

**5.** [Building an ASP.NET MVC sitemap provider with security trimming](/post/2008/08/29/building-an-aspnet-mvc-sitemap-provider-with-security-trimming.aspx) – Building an ASP.NET MVC sitemap provider that can be used with standard ASP.NET sitemap controls.

## About the past year

Though 2008 seems to have been a year in which I mostly blogged on the [ASP.NET MVC framework](/category/MVC.aspx), the top 3 posts seem to be on PHP and ASP.NET. Still, 2 posts on ASP.NET MVC in my top 5.

The past year brought lots of attention to my [PHPExcel](http://www.phpexcel.net) and [PHPLinq](http://www.phplinq.net) project, both from the PHP side ([Zend](http://www.zend.com/)) as well as the [Microsoft](http://www.microsoft.com) side. Both projects will continue evolving in 2009, watch out for a huge post on all new features in PHPLinq in January. By the way, I did not blog about a great milestone PHPExcel made: PHPExcel has been downloaded over 120.000 times!

[![](/images/WindowsLiveWriter/Topblogpostsin2008_D71D/image_thumb.png)](/images/WindowsLiveWriter/Topblogpostsin2008_D71D/image_2.png)

Blogging about ASP.NET MVC ever since Microsoft started working on that framework also brought me and my blog a lot of attention. The [CarTrackr sample application](http://www.cartrackr.net) I created for a [VISUG session in Belgium](/post/2008/10/15/Introduction-to-ASPNET-MVC-for-VISUG-Presentation-materials.aspx) has also been downloaded over 800 times.

My blog had 115.988 visits in the past year, with currently around 400 unique visits a day (not counting RSS readers of which I don’t have any stats unfortunately). Keep spreading the word, keep [subscribing to my RSS feed](/syndication.axd).

## Looking ahead at 2009

2009 will bring more good stuff: continued development on PHPExcel and PHPLinq, more ASP.NET MVC, perhaps some more on Azure or any other new Microsoft technology. Also expect a book on the ASP.NET MVC framework with [Packt Publishing](http://www.packtpub.com/), named “ASP.NET MVC Quickly”. It’s currently in technical review and should be available somewhere near end January.

Happy new year 2009 to all my readers! See you around next year.
