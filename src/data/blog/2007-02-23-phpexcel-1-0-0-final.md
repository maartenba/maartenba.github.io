[![](/images/WindowsLiveWriter/PHPExcel1.0.0final_A301/office2007_logo_thumb%5B1%5D.gif)](/images/WindowsLiveWriter/PHPExcel1.0.0final_A301/office2007_logo%5B1%5D.gif) I'm quite proud to announce the [1.0.0 version of PHPExcel](http://www.codeplex.com/PHPExcel), a set of classes that allows you to wirte Excel2007 files from PHP. Since my last post on this, a lot of things happened, feature-wise and project-wise.

Feature-wise, conditional formatting was added, and a first attempt to a Excel2007 file reader has been created. Project-wise, Jakub Vrana signed up as a developer. He's currently working on an improved version of the Excel2007 Reader.

We are still looking for extra developers: if you know PHP 5, object-oriented programming, have knowledge of/interest in OpenXML or the Open Document Format (as we plan to include that too), [please apply](http://www.codeplex.com/PHPExcel).
